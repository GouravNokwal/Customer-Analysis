# Customer-Analysis
A data analysis project that explores customer behavior, purchasing patterns, and key business insights using Python, Pandas, and Matplotlib. The project focuses on data cleaning, exploratory data analysis (EDA), and visualization to support data-driven decision making.

# Data Analytics Project

## Project Overview
This project demonstrates the complete Data Analytics workflow, from loading and cleaning data to analyzing it and creating a visual dashboard.  
The goal is to extract meaningful insights from the dataset and present them clearly using Python, SQL, and Power BI screenshots.

---

## Objectives
- Load and explore the dataset
- Perform Exploratory Data Analysis (EDA)
- Clean and preprocess the data
- Run analytical queries using SQL
- Present key insights visually through dashboard screenshots

---

## Tools and Technologies Used
- Python (Pandas, NumPy)
- SQL (PostgreSQL)
- Power BI (Free Desktop for creating dashboard images)
- JupyterLab

---

## Data Analysis Steps

### 1. Data Loading
The dataset was loaded into Python using Pandas for analysis.

### 2. Exploratory Data Analysis (EDA)
EDA was performed to understand:
- Data distribution
- Missing values
- Relationships between variables
- Key trends

### 3. Data Cleaning
The following preprocessing steps were applied:
- Handling missing values
- Removing duplicates
- Correcting data types
- Filtering irrelevant records

### 4. SQL Analysis
SQL queries were used to perform deeper analysis such as:
- Aggregations
- Grouping
- Filtering
- Calculating business metrics

### 5. Dashboard
A Power BI dashboard screenshot is included to visualize insights such as:
- Sales trends
- Regional performance
- Profit analysis
- Category-wise comparison

---

## Key Insights
- Identified top performing categories
- Found regions generating the highest revenue
- Observed seasonal trends in sales
- Highlighted areas with low profitability
